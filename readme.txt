A simple Todo App plugin with React frontend.

== Description ==

Todo App Plugin provides a simple, elegant task management interface built with React, Vite, TailwindCSS, and Shadcn UI components.

Features:
* Add new tasks
* Mark tasks as complete
* Delete tasks
* Clean and modern UI with Shadcn components

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/todo-app-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the 'Todo App' menu item in your admin sidebar to access the todo application

== Frequently Asked Questions ==

= Does this plugin store tasks in the database? =

No, this is a client-side only app. Tasks are stored in the browser's local storage and will persist across page refreshes, but are not saved to the WordPress database.
